Help Page

From the Read Page Lower Toolbar, tap the Help Page button to open the Help Page.

The Help Page has one Subpage: Topic.

The Help Page Upper Toolbar displays the name of the Active Topic.

The Help Page presents help information for the Active Topic.

To cancel, on smaller devices, tap the Back button to return to the Read Page.

Topic Subpage

Tap the Topic button in Help Page Upper Toolbar to open the Topic Subpage.

The Topic Page displays a list of help topics.

Tap a Topic name to set the Active Topic and open the Help Page.

To cancel, tap the Back button to return to the Help Page.
