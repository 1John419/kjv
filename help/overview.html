Design

This is an offline application, which means that once downloaded, an internet connection is not required to use it.

The application supports any Android, Windows, or Linux platform with the latest version of Google Chrome or Mozilla Firefox web browser. (It should also support the Safari web browser, but has not been tested on Mac/iOS platforms.)

On an Android platform, the best user experience is available using Chrome for Android. By selecting 'Add to Home screen', the application mimics a native Android app.

Navigation And Icons

The application uses six main Pages: Read, Book, Chapter, Bookmark, Search, Setting, and Help.

The Bookmark, Search, and Help Pages have additional Subpages.

All Pages have an Upper Toolbar and a Lower Toolbar. These provide context information and buttons to access other features.

Use the Back button in the Lower Toolbar to return to the previous Page:

Back

On smaller devices, only one Page is visible at a time. Tap the desired button to display that Page or Subpage. Tap the Back button to return to the Read/Previous Page.

On larger devices, the Read Page is always visible and a single sidebar displays one of the other Pages. Tap the corresponding button on the Read Page to display the desired Page in the sidebar.
