Setting Page

From the Read Page Lower Toolbar, tap the Setting Page button to open the Setting Page.

The Setting Page allows the user to select preferred application options.

A sample of scripture text is displayed to reflect options selected.

Font

Several fonts are available for verse text. Tap the Previous/Next button to scroll through available fonts. The selected font name is displayed.

Font Size

Several font sizes are available for verse text. Tap the different Font Size buttons to select the preferred size.

Theme

Several application color themes are available. Tap the Previous/Next button to scroll through available themes. The selected theme name is displayed.
