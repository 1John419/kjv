Read Page

The Read Page displays the verses of the Active Chapter.

The Active Chapter is set via the Previous/Next Chapter buttons or the Book, Chapter, Bookmark, or Search Page.

For more details, see the Book/Chapter, Bookmark, and Search Help Topic.

Tap the Single, Double, or Triple Column button (if present) to set the verse display mode.

Bookmarks

Bookmarks in the Active Folder for the Active Chapter are highlighted on the Read Page.

Tap on a verse's text to add/remove a Bookmark in the Active Folder.
